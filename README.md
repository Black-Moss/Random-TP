# Random TP

Teleports you to a random location every 60 seconds(default).

Use the `randomtp` command **instantly** random tp and reset the cooldown!

__Requires [Moss Lib](https://github.com/Black-Moss/Moss-Lib) as a prerequisite!__

Configs `blackmoss.randomtp.cfg`:

* `TpCountdown`: The interval between every time tp, default: 60f
* `TpTipStyle`:
    * true: Center of the screen
    * false: Bottom of the screen

# 随机传送

在60秒（默认）内随机传送。

使用`randomtp`命令**立即执行**一次随机传送，同时重置随机传送冷却！

__需要[Moss Lib](https://github.com/Black-Moss/Moss-Lib)作为前置！__

配置文件 `blackmoss.randomtp.cfg`:

* `TpCountdown`: 每次传送的间隔，默认: 60f
* `TpTipStyle`:
    * true: 屏幕中间
    * false: 屏幕底部
